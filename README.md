# Delaunay3D

Delaunay diagram in 3D for Delphi.

四面体メッシュ（TetraFlip）の上に築いた、逐次添加法による 3D ドロネー図のサンプルです。
点の追加（Bowyer-Watson 法）と削除（InSphere 誘導のフリップ法）に対応し、FMX の 3D シーンで
ドロネー四面体とボロノイ辺を表示します。[Delaunay2D](https://github.com/LUXOPHIA/Delaunay2D) の
設計（無限遠頂点方式・統一リフト述語・TFrame ビューア）を 3D に拡張したものです。

## 構成

```
Main.pas / Main.fmx                  … アプリケーション（薄いフォーム。シーン生成コードは持たない）
_LIBRARY\
  LUX.Delaunay.D3.pas                … 3D ドロネー図（点の追加・削除・探索）
  LUX.Delaunay.D3.Viewer.pas/.fmx    … ビューア（TFrame。FMX 3D のシーンをここへ集約）
  LUXOPHIA\LUX\
    Data\Model\TetraFlip\            … 四面体メッシュのデータ構造（実装層・型付け層・3D特殊化）
    …                                … LUX 基盤（ベクトル・リスト・モデル基底）
Tests\Delaunay3DTest.dpr             … 検証プログラム（構造・向き・空球性・挿入削除の混合試験）
```

## モデル

- **TetraFlip** … 胞（四面体）は4つの頂点 `Poin[0..3]` と、面ごとの隣接情報
  `Cell / Corn / Bond`（隣の胞・その対頂点番号・共有面の回転コード）を持つ。
  回転コードは1バイトに 2bit × 4 で詰め込まれ、巡回表 `VertTable / BondTable` との合成
  `Join[K,I]` が面の頂点対応を与える。縫合は頂点の同一性から回転コードを導出する
  `Weld` の一手のみ。
- **無限遠頂点** … スーパーテトラの代わりに、ただ一つの無限遠頂点が凸包の外側を
  「無限遠胞」で覆う。リフト（`0,0,0,1`）と内外判定を多態で差し替えるため、
  述語にフラグの分岐は存在しない。球と平面、外心と外向きの方向は、同じ式の中で
  自然に退化しあう。

## アルゴリズム

- **追加** … Bowyer-Watson 法。追加点を外接球に含む胞群（キャビティ）を再帰
  `CellTree` で削除し、境界面ごとに新しい胞を張り直す。3D ではキャビティの双対が
  木にならないため、再突入をプレースホルダ胞（縫合待ちの受け箱）で解決する。
  検索はジャンプ＆ウォーク（期待 O(n^(1/4))）。
- **削除** … フリップ法。頂点の星の耳（隣接2胞の張る四面体）のうち「自分が耳の
  外接球の内側にあり、球が他のリンク頂点を含まない」もの（＝最終形に現れる胞）を
  2-3 / 3-2 フリップで確定していき、星が4胞になったら1胞に畳み込む。穴を開けない
  ため途中状態が常に正しい四面体分割であり、凸包上の頂点も同じ手順で削除できる。
  反射辺の環が閉じていれば 3-2 を優先して複製を防ぎ、凸包の周りで生じる体積ゼロの
  鏡像対（ポケット）は 2-0 フリップで除去する。フリップで畳み切れない退化配置
  （数%）は、点のインスタンスを保ったまま胞だけを作り直して確実に取り除く。

## 操作

- **左ドラッグ** … 回転　/　**ホイール** … ズーム
- **クリック** … 近くの点を削除
- **Add x10 / Del x10 / Clear** … 点のランダム追加・削除・全消去
